# Pokedux

Um web app em React para encontrar Pokemons.


## Stack
- React
- Hooks
- Router
- ContextAPI
- Typescript
- AXIOS
- Styled Components
- Jest


## Running

```
npm start
```


## Unit Testing

```
npm test
```


## Structure

```
src
├── components
│   ├── pages
│   └── ui
├── configs
└── services
```
